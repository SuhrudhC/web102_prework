## Technical Skills Demonstrated

- **DOM Manipulation**: `createElement()`, `appendChild()`, `classList.add()`, `innerHTML`
- **Array Methods**: `filter()`, `reduce()`, `sort()`, destructuring with spread operator
- **Event Handling**: Event listeners for filter buttons
- **Template Literals**: Dynamic HTML content generation with variables
- **ES6+ Features**: Arrow functions, destructuring, spread operator, const/let
- **CSS**: Flexbox layouts, hover effects, responsive design

## Notes

### Challenges Encountered and Solutions:

1. **Understanding reduce()** - Initially struggled with the accumulator concept, but the "donation bucket" metaphor helped clarify how values accumulate across iterations.

2. **Destructuring and Spread Operator** - Learning the difference between spreading array elements versus accessing specific indices required practice with examples.

3. **Ternary Operators for Grammar** - Implementing conditional text for singular/plural required careful planning to ensure grammatically correct output.

4. **CSS Flexbox Alignment** - Getting the stats cards to align properly with flexbox required understanding both `display: flex` and `align-items` properties.

5. **Event Listener Management** - Ensuring the deleteChildElements() function was called before displaying filtered results was crucial to prevent duplicate game cards from appearing.

## Installation & Usage

1. Clone the repository
2. Open `index.html` in a web browser
3. Interact with the buttons to filter games by funding status
4. View statistics and the top 2 most funded games at the top of the page
